Veille technologique

Virtualisation & Cloud

Suivi des évolutions technologiques autour de la virtualisation et des infrastructures cloud, dans le cadre du BTS SIO SISR.

☁️
Pourquoi ce sujet ?

La virtualisation et le cloud transforment profondément les infrastructures IT des entreprises. En tant que futur administrateur SISR, j'ai déjà manipulé Hyper-V et Proxmox en formation. Comprendre les évolutions vers le cloud hybride et les technologies de conteneurisation est devenu indispensable pour mon profil.

🖥️
La Virtualisation

Permet de faire tourner plusieurs OS sur une seule machine physique via un hyperviseur :

  • Type 1 (bare-metal) — directement sur le matériel : Hyper-V, Proxmox, VMware ESXi
  • Type 2 (hosted) — sur un OS existant : VirtualBox, VMware Workstation
Hyper-V Proxmox VMware ESXi VirtualBox
🌐
Le Cloud Computing

Ressources informatiques à la demande via Internet. Trois modèles de déploiement :

  • Cloud public — mutualisé : AWS, Azure, Google Cloud
  • Cloud privé — infrastructure dédiée à une organisation
  • Cloud hybride — combinaison des deux, tendance dominante
AWS Azure GCP OVHcloud
⚙️
Modèles de service

Trois niveaux selon la responsabilité partagée :

  • IaaS — VMs, stockage, réseau (ex : Azure VM)
  • PaaS — environnement de dev géré (ex : Azure App Service)
  • SaaS — app directement utilisable (ex : Microsoft 365)
IaaS PaaS SaaS
Tendances & évolutions récentes
🐳
Conteneurisation

Docker et Kubernetes s'imposent comme standards pour déployer des apps portables et scalables, en complément de la virtualisation classique.

🔒
Sécurité Cloud

Chiffrement, IAM, conformité RGPD et segmentation réseau sont devenus prioritaires dans toute architecture cloud moderne.

🏢
Infrastructure hybride

La majorité des entreprises combinent datacenter on-premise et services cloud pour maximiser flexibilité et continuité de service.
